# FTIB — Fashion Trend Intelligence Board

> 패션 트렌드의 기원(Origin)과 전파 경로를 데이터 기반으로 추적·분석하는 인텔리전스 대시보드
> 최종 업데이트: 2026-03-16

---

## 1. 프로젝트 소개

### 한 줄 요약
런웨이 → 전문가 → 셀럽 → 마켓으로 이어지는 패션 트렌드 전파 과정을 정량적으로 추적하여, 상품 기획·바잉 의사결정에 데이터 근거를 제공하는 대시보드.

### 핵심 가치
- **감을 대체하는 도구가 아니라, 감의 근거를 제공하고 블라인드 스팟을 잡아주는 도구**
- "기능을 만들고 맞는지 확인"이 아니라, **"방법론을 검증하고 검증된 것만 구현"**

### 대상 사용자
- F&F AX팀 (상품 기획, 바잉, 트렌드 분석 담당)

### 기술 스택
| 영역 | 기술 |
|------|------|
| Backend | FastAPI + Python 3.9 + SQLite (aiosqlite) |
| Frontend | React 18 + Vite + Tailwind CSS |
| Charts | Recharts, react-force-graph-2d |
| Crawling | Playwright (마켓), requests+BS4 (TagWalk 런웨이) |
| VLM | Claude Vision Sonnet (이미지 라벨링) |
| Deploy | GitHub + Vercel (서버리스) |

### 라이브 URL
🔗 https://fashion-trend-board.vercel.app

## 2. 핵심 프레임워크: Trend Origin 분류

### 대전제
> "패션 트렌드는 다양한 기원(Origin)에서 출발하여 각기 다른 경로로 전파되며, 기원에 따라 속도·강도·지속성이 다르다."

### 4가지 Origin 타입

| Origin | 시그널 플로우 | 특징 | 딜레이 |
|--------|-------------|------|--------|
| **Runway-led** | 런웨이 → 전문가 → 셀럽 → 검색 → 마켓 | 순차적, 예측 가능 | 6~12개월 |
| **Capital-driven** | 브랜드 투자 → 앰배서더 캠페인 → 검색 폭발 → 마켓 | 런웨이 건너뜀 | 1~3개월 |
| **Viral/Meme** | 소셜 밈 → 틱톡 확산 → 검색 급등 → 마켓 소진 | 예측 불가, 단명 | 2~4주 |
| **Market-organic** | 소비자 실수요 → 점진적 확대 → 검색 상승 | 선행 시그널 없음 | 점진적 |

### 패션 조닝별 Origin 지배 구조

| 조닝 | Runway | Capital | Viral | Organic |
|------|--------|---------|-------|---------|
| 럭셔리/하이엔드 | **60%** | 25% | 10% | 5% |
| 스포츠/아웃도어 | 15% | 30% | 15% | **40%** |
| 캐주얼/스트리트 | 20% | 30% | **35%** | 15% |
| SPA/매스 | 25% | 20% | 30% | 25% |

## 8. 검증해야 할 핵심 가설

| # | 가설 | 검증 방법 | 상태 |
|---|------|----------|------|
| H1 | 런웨이-마켓 간 유의미한 상관관계 존재 | 런웨이 Top 30 vs 마켓 전수 매칭 | ⬜ Phase 0 |
| H2 | 전문가 리포트에 독립적 예측력 존재 | 풀 A vs 풀 B 적중률 비교 | ⬜ Phase 1 |
| H3 | 셀럽 착용은 마켓 확산의 선행 지표 | 검색량 스파이크 → 마켓 시차 분석 | ⬜ Phase 2 |
| H4 | 시즌 간 변화로 트렌드 방향 판별 가능 | VLM 시각 속성 시즌 비교 | ⬜ Phase 0 |
| H5 | Origin 타입에 따라 전파 패턴 구분 가능 | 시그널 발생 순서 기반 자동 분류 | ⬜ Phase 2 |

### "적중" 정의 (3단계)

| 레벨 | 기준 | 측정 방법 |
|------|------|----------|
| L1 검색 관심 | 네이버 DataLab 주간 검색량 +30% | Naver DataLab API |
| L2 마켓 반영 | 키워드 포함 마켓 상품 수 증가 | 반복 크롤링 + 매칭 |
| L3 판매 검증 | 품절률·판매 순위 상승 | 크롤러 재고 상태 추가 |
